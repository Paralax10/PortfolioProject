Project Overview

This project is a speed game where the player must press the correct button when the LED lights up. There are a total of four LEDs that light up in sequence, and the player has to react quickly. The game speeds up over time, making it more challenging. The project used the Arduino platform, and the programming was done in C++.

Key Features of the Speed Game

  • Multiple LED lights to create a dynamic gameplay experience
  • Game logic that increases difficulty as the game progresses
  • Button-based user input for responsive interactivity
  • Real-time reaction tracking and score display using a 7-segment display
  • Custom circuit assembly using LEDs, resistors, and microcontrollers
